# ملفي الشخصي - My Profile

## نظرة عامة
صفحة الملف الشخصي للموظف حيث يمكنه عرض وتحديث بياناته الشخصية والوظيفية.

## المكونات الرئيسية

### 1. البيانات الشخصية
- الاسم الكامل
- الصورة الشخصية
- معلومات الاتصال (الهاتف، البريد الإلكتروني)
- العنوان
- تاريخ الميلاد
- الحالة الاجتماعية

### 2. البيانات الوظيفية
- المسمى الوظيفي
- القسم
- الإدارة
- المدير المباشر
- تاريخ التعيين
- رقم الموظف

### 3. المعلومات المالية
- رقم الحساب البنكي
- الراتب الأساسي
- البدلات

### 4. معلومات الطوارئ
- جهة الاتصال في حالات الطوارئ
- رقم الهاتف للطوارئ
- العلاقة

## الميزات

- **عرض المعلومات**: عرض جميع البيانات الشخصية والوظيفية
- **تحديث البيانات**: إمكانية تحديث بعض البيانات القابلة للتعديل
- **تحميل المستندات**: رفع نسخ من المستندات الشخصية
- **سجل التغييرات**: عرض سجل التحديثات على البيانات

## الصلاحيات

- **الموظف**: عرض وتعديل البيانات الشخصية فقط
- **المدير**: عرض بيانات المرؤوسين
- **الموارد البشرية**: الوصول الكامل لجميع البيانات

## الاتصال بالأنظمة الأخرى

- **نظام الحضور**: ربط البيانات الشخصية بسجل الحضور
- **نظام الرواتب**: ربط البيانات المالية
- **نظام التدريب**: ربط البيانات المهنية

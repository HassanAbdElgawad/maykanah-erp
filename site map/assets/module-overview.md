# نظرة عامة - وحدة الأصول

## الغرض
إدارة شاملة للأصول الثابتة والمتداولة مع متابعة حركاتها وصيانتها.

## المكونات الرئيسية

1. **إدارة الأصول** - قائمة وإدارة جميع الأصول
2. **تعديل قيمة الأصل** - تعديل القيم والإهلاك
3. **الصيانة** - جدولة ومتابعة الصيانة
4. **حركات الأصول** - تسجيل النقل والاستخدام
5. **البيع أو الاستبعاد** - عمليات التخلص من الأصول

## التكامل
- الربط مع المحاسبة
- الربط مع المخازن
- الربط مع المشتريات
